# MoneyKawKaw


## Run the project
1. Navigate to the app directory:
```bash
cd app
```

2. Install dependencies:
```bash
npm install
```

3. Install Expo dependencies:
```bash
npx expo install
```

4. Start the development server:
```bash
npx expo start
```

5. Open the app using either:
   - Android Studio emulator: press "a" in the terminal
   - Mobile device: scan the QR code in the terminal using the Expo Go app
